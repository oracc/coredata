Installing the ESP templates and schema

Before you begin

Will you be managing the whole ESP portal site yourself, or are you just writing ESP pages that someone else will edit and upload?

In either case you will need to:

  • download and install a copy of Emacs on your computer;
  • install a set of schemas on your computer that Emacs will use automatically to ensure that your files are valid.

If you are managing the ESP portal site yourself on Oracc, you will also need to:

  • initiate ESP in your Oracc project, to install the necessary templates there.

How to install the ESP schemas on your computer

  • Download this zip file: oracc-schemas.zip [~/downloads/oracc-schemas.zip].

  • Create a directory (folder) called, say, oracc, in the directory where your .emacs file lives.

  • Unzip oracc-schemas.zip into the directory you just created (e.g., oracc).

  • Open your .emacs file and add this line of code to it:

    (eval-after-load 'rng-loc '(add-to-list 'rng-schema-locating-files
 "~/oracc/schemas/schemas.xml"))

    (Note that if you chose a different name/place for what we are calling the oracc directory, you should replace ~/oracc with the directory name you chose.)

  • Save your .emacs file, quit Emacs and restart.

How to initiate ESP on the Oracc server

You should only do this if you are managing the ESP portal site yourself on Oracc. If you are just writing ESP pages for someone else to install you don't need to do this.

  • At the top level of your project run the Oracc command oracc esp init. This installs all the templates you will need in your 00web/esp/ directory.

You only need to do all of this once; now you are ready to use ESP.
