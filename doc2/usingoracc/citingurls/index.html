Citing Oracc URLs online and in print

This page describes how to refer to Oracc web pages online or in print, using short, stable URLs. A wider range of options, described in more technical language, is given on the Oracc URI patterns page.

Online or In Print?

When linking to Oracc from a web page or another online resource, just follow the instructions below.

When citing Oracc resources in print, you should not cite the oracc.museum.upenn.edu server directly. Instead, use the hostname http://oracc.org. Then append the relevant part of the URL as given in the examples, which are designed to be permanent. The hostname http://oracc.org will always redirect to the current Oracc server, wherever it is hosted.

Projects

Decide whether you want to link to the home page of a project (which is likely to include a text-based description of its aims and scope), or directly to the corpus outline. Always the lower-case project abbreviation, and remember to include the final /. For instance:

Project home page in print
http://oracc.org/dccmt/
Project home page online
http://oracc.museum.upenn.edu/dccmt/
Corpus home page in print
http://oracc.org/dccmt/corpus/
Corpus home page online
http://oracc.museum.upenn.edu/dccmt/corpus/

Some projects have subprojects:

Subproject home page in print
http://oracc.org/cams/gkab/
Subproject home page online
http://oracc.museum.upenn.edu/cams/gkab/
Subproject corpus home page in print
http://oracc.org/cams/gkab/corpus/
Subproject corpus home page online
http://oracc.museum.upenn.edu/cams/gkab/corpus/

If a project (or subproject) does not have a separate home page, the home-page URL defaults to the corpus home page. For instance, http://oracc.org/contrib/amarna/ is equivalent tohttp://oracc.org/contrib/amarna/corpus/.

Texts

There are various different ways of linking to single or several texts in a corpus. For instance:

Single text

To link directly to a single text in a project, for instance to imbed it in another web page, use the the text's ID (i.e., P-, Q- or X-number) and add /html. For example:

  • http://oracc.museum.upenn.edu/dcclt/Q000001/html
  • http://oracc.museum.upenn.edu/cams/gkab/P363523/html

We don't recommend this method for print citations.

List of texts

There are a few different ways of generating a list of one or more texts in a project:

Using text IDs
List the IDs P-, Q- or X-number(s) ) of one or more texts, separated by commas (but no spaces) and no final .html or /. For example:
  • http://oracc.museum.upenn.edu/dcclt/Q000001,Q000002
  • http://oracc.museum.upenn.edu/dcclt/Q000003
  • http://oracc.org/cams/gkab/Q002715,P363442,P363443
Using keywords from the catalogue
Sometimes it is useful to provide a list of texts as if they were the results of a catalogue search. Use a question mark ? before the search term. For instance:
  • http://oracc.museum.upenn.edu/qcat?sargon lists all compositions in the Q-catalogue that have "Sargon" in the title, whether Sargon the Great or Sargon II;
  • http://oracc.org/saao?query lists all the divinatory queries in State Archives of Assyria online;
  • http://oracc.museum.upenn.edu/cams/gkab?iqqur-ipuš lists all the manuscripts and Tablets of Iqqur Ipuš in CAMS/GKAB. Note the hyphen instead of the space.

Glossaries

Whole glossaries

Give the lower-case abbreviation for the glossary or subglossary, without a final /. For example:

  • http://oracc.museum.upenn.edu/cams/akk for the top-level CAMS Akkadian glossary;
  • http://oracc.museum.upenn.edu/etcsri/qpn-x-divine for ETCSRI's Divine Names subglossary of the Proper Nouns;
  • http://oracc.museum.upenn.edu/saao/saa01/akk-x-neoass for the Neo-Assyrian subglossary of the Akkadian glossary of State Archives of Assyria online, volume 1.

Glossary entries

You can also give lists of one more glossary entries, using Sumerian, Akkadian and/or English keywords. For instance:

  • http://oracc.museum.upenn.edu/cams/gkab/akk?adru lists the entries for both adru [AFRAID] AJ and adru [DARK] in CAMS/GKAB's Akkadian glossary;
  • http://oracc.museum.upenn.edu/cams/gkab/akk?afraid lists all the entries for Akkadian words that mean "afraid" in CAMS/GKAB's Akkadian glossary;
  • and http://oracc.museum.upenn.edu/cams/gkab/akk/adru[afraid]AJ lists only the entry for adru [afraid] AJ.

Note the akk/ instead of akk? in that last example.

Further information

The range of citation patterns, described in more technical language, is given on the Oracc URI Patterns page.
