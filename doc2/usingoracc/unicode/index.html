Unicode characters for Cuneiform transliteration

This page lists the essential Unicode characters for transliterating and normalising Sumero-Akkadian cuneiform.

Several families of fonts containing all of these characters are available from the fonts page.

A collection of keyboards supporting Unicode transliterations implements a consistent set of input conventions across operating systems. The keystrokes in the 'KEYS' column below are composed of the comma character (,) followed by the character typed to produce a Unicode result.

For complete details please see the keyboards page.
