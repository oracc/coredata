Access Keys

Most browsers let you jump to specific pages using access keys defined by a web site. On Windows, you hold down the Alt key and press the access key. On a Mac, you use Ctrl in the same way. On some browsers, you then press Return or Enter.

This site defines the following access keys:

KeyPage
0Access Keys
1Documentation home
3Site Map
4Search site (move to search box on current page)
6Editing in ATF
7Licensing
8Decrease text size
9Increase text size
SSkip to main content (on current page)
